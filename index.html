      W3 Coding Best Practice
    • 1. Always declare '!doctype html', 'lan="en"', viewport with device-width and 'meta charset="utf-8"' unless you have a reason not to
    • 2. Proper naming: lowercase, split with '-' and tag your elements
    • 3. Always close elements, even if not necissarily needed
    • 4. Qoute attribute values
    • 5. Always specify alt, width and height for images
    • 6. Focus on readability: no long lines, appropriate indentation
    • Use "li" to list items encompassed within "ul" tag which stands for list and unordered list respectively
